Welcome to the dotnet-k8s-stack (github)!

Each of the pages on the upper menu will demonstrate the capabilities of Kubernetes and provide an example to how to implement a 3-tier application consisting of a frontend SPA, backend API, and database. This application uses Angular, .NET Core, and Microsoft SQL, but other stacks are just as possible because of the containerization nature of the entire application.

This current implementation is deployed to Azure Cloud and Kubernetes is managed with Azure Kubernetes Service (AKS). However, with Infrastructure-as-Code (Terraform) we can deploy this Kubernetes stack to other cloud providers as well. The goal of this project is to show how Kubernetes allows for cloud-agnostic implementations.
